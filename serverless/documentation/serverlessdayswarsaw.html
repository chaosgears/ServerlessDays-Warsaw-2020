/registration

A collection of guest registration

/registration post

post

Add a new guest registration and send him a confirmation email
with a link through which the registration will be deleted.

post /registration

Add a new guest registration and send him a confirmation email
with a link through which the registration will be deleted.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • firstName: required(string)

    First name

    Example:

    Jan
  • lastName: required(string)

    Last name

    Example:

    Kowalski
  • email: required(string)

    Email

    Example:

    jan.kowalski@softgame.com
  • position: required(string)

    Position in the organization

    Example:

    Junior Software Developer
  • organizationName: required(string)

    Name of the organization

    Example:

    SoftGame
  • businessInterests: required(list)

    Example:

    [
  "Test"
]
  • technicalInterests: required(list)

    Example:

    [
  "Test"
]
  • isCanceled: required(boolean)

    Property is set to true if the guest cancels registration, otherwise the flag is false

    Example:

    True

Example:

{
    "firstName": "Jan",
    "lastName": "Kowalski",
    "email": "jan.kowalski@company.com",
    "position": "Junior Software Developer",
    "organizationName": "SoftGame",
    "businessInterests": ["Test"],
    "technicalInterests": ["Test"]
}

The output contains the unique identifier of the newly created registration.

Body

Media type: application/json

    "message": "Success",
    "output": "520d6254-72b2-48c3-a2f2-44bd1e7d3be3"
}

Type: any

HTTP status code 409

Body

Media type: application/json

Type: any

Example:

    "output" : "520d6254-72b2-48c3-a2f2-44bd1e7d3be3"

/presentation

A collection of presentations

/presentation post

post

Add a new presentation request and send a thank you email to the author. Author must be registered as an event guest.

post /presentation

Add a new presentation request and send a thank you email to the author. Author must be registered as an event guest.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • email: required(string)

    Email of the author

    Example:

    jankowalski@softgame.pl
  • presentationTitle: required(string)

    Title of presentation

    Example:

    AWSome techniques to build your app!
  • presentationDescription: required(string)

    Description of presentation, maximum length is 400 characters.

    Example:

    In my presentation I want to show some great tricks to youngsters  how to efficiently build their first application.

Example:

{
    "email": "jan.kowalski@softgame.pl",
    "presentationTitle": "AWSome techniques to build your app!",
    "presentationDescription": "In my presentation I want to show some great tricks to youngsters 
        how to efficiently build their first application."
}

HTTP status code 200

The output contains the unique identifier of the message.

Body

Type: any

Example:

    "message": "Email sent! Message ID: 12345",
    "output": "12345"
